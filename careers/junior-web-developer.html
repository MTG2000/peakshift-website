Junior Web Developer

Introduction

We're looking for a web developer to build out projects for our clients and internal app. You will be working closely with our head developer/designer.

Some of our projects are based on PHP and Open Source systems like WordPress. We are also in the process of expanding more into mobile design and development.

This is a remote job, you get to work from home or where-ever you choose. You will be engaged with at least one full project at anytime but may also be required to assist on parts of another project going on.

The Role

A typical project you would be assigned can start in two ways:

  1. With you converting a design from Designs to HTML and CSS and use Javascript where needed for scripting.
  2. Setting up the backend system’s database, site structure and building out required functionality before it gets hooked up to the front-end templates.

Skills Required

  • Javascript
  • Skills to convert a UI Designs to vanilla HTML and CSS
  • Some Knowledge of PHP & MySQL
  • Working Knowledge of WordPress
  • Ability to communicate effectively in English

Additional skills with Node.js, Client Side MV* (we use Vue.js) and Native Applications would be a huge plus!

Git for version control, SASS and Bower as component manager to name a few.
