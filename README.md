# README #



## Getting Started

### Prerequistes:

- node
- npm
- sass


### Setup: 

- run `npm install -g http-server` if you don't have http-server installed already
- clone repository

### Compile and Run:

- run `npm start` to get the development server running
- run `npm run sass` to watch the sass file for changes and automatically compile the css
